1. Justificación del Framework

¿Para qué sirve? Angular es un framework para construir aplicaciones web de una sola página (SPA) y aplicaciones móviles.

¿Qué aplicaciones tiene? Se utiliza para desarrollar aplicaciones web interactivas, aplicaciones móviles híbridas y aplicaciones empresariales complejas.

¿Sobre qué lenguajes se apoya? Angular está basado en TypeScript, un superconjunto de JavaScript.

¿Qué prerrequisitos necesita? Conocimientos básicos de HTML, CSS y JavaScript. Familiaridad con TypeScript es útil pero no esencial.

2. Instalación

¿Se precisa instalación? Sí, es necesario instalar Angular CLI.

Pasos para la instalación:

  1. Instalar Node.js desde nodejs.org.
  2. Instalar Angular CLI: npm install -g @angular/cli
  3. Crear un nuevo proyecto: ng new nombre-proyecto
  4. Navegar al directorio del proyecto: cd nombre-proyecto
  5. Iniciar el servidor de desarrollo: ng serve

Versiones necesarias: Node.js LTS, Angular CLI.

Rutas para tener en cuenta: Asegúrate de que Node.js y npm estén en el PATH del sistema.

Variables de entorno: En la sección "Variables del sistema", busca y selecciona la variable Path, luego haz clic en "Editar". Añade la ruta de instalación de Node.js. Por defecto, suele ser C:\Program Files\nodejs\.

¿Necesita base de datos? No, pero puede integrarse con bases de datos si es necesario.

3. Primeros Pasos

Cómo realizar una primera aproximación (“Hola Mundo”):

  1. Abre el archivo src/app/app.component.html.
  2. Reemplaza el contenido con: <h1>Hola Mundo desde Angular</h1>
  3. Guarda los cambios y observa la actualización en el navegador.

4. Utilización del Framework

¿Cómo empezar a crear una aplicación base?

  1. Generar un nuevo componente: ng generate component nombre-componente
  2. Editar el componente: Abre src/app/nombre-componente/nombre-componente.component.html y añade contenido HTML.
  3. Visualizar los resultados: Asegúrate de que el componente se incluye en app.component.html para verlo en la aplicación.

5. Explicación del Funcionamiento del Framework

¿Actúa sobre páginas, ficheros, secciones? Angular organiza la aplicación en módulos y componentes. Cada componente tiene su propio HTML, CSS y lógica en TypeScript.

¿Cómo se editan/modifican los ficheros? Los ficheros se editan directamente en el editor de código. Angular CLI facilita la creación y gestión de estos ficheros.

¿Cómo se referencian los ficheros creados desde otras aplicaciones? Los componentes y servicios se importan y utilizan en otros módulos y componentes mediante el sistema de módulos de TypeScript.

6. Conclusiones

Al finalizar este tutorial, habrás aprendido a instalar Angular, crear una aplicación básica y entender los conceptos fundamentales del framework. Este conocimiento te permitirá desarrollar aplicaciones web dinámicas y escalables.
